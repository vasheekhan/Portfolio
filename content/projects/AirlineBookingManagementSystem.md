Designed and developed a distributed airline booking platform powered by a microservices architecture. Built independent services for authentication, flight management, booking, reminders, and API Gateway while implementing secure JWT authentication, event-driven communication with RabbitMQ, Redis caching, and transactional booking workflows to create a scalable and production-ready backend system.
