A production-inspired airline booking platform built using a scalable microservices architecture. The system consists of dedicated services for authentication, flight search, booking management, reminders, and an API Gateway for centralized request routing. It leverages JWT authentication, RabbitMQ for asynchronous communication, Redis caching, and MySQL with Sequelize to deliver secure, reliable, and high-performance backend operations.
